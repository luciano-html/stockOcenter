# Decisión SDD #06 — UI Library

## Pregunta 6.1: Librería de componentes
- **Opción A:** shadcn/ui (TailwindCSS)
- **Opción B:** Material UI (MUI)
- **Opción C:** Ant Design
- **Opción D:** TailwindCSS puro

**Decisión:** A — shadcn/ui

**Justificación:** Es la librería más moderna del ecosistema React + TailwindCSS. Componentes accesibles, personalizables, y livianos. Ideal para mobile-first por su flexibilidad con Tailwind.
